# vue-demo-or-test
此项目用来个人编码小的vue demo，测试vue的组件，学习vue用法等
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
