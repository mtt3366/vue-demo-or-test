# vue-demo-or-test
此项目用来个人编码小的vue demo，测试vue的组件，学习vue用法等
## 添加一个demo的方式
复制`views`文件夹下的`TemplateDemo`文件夹,并且自定义文件夹的名字,然后在`router`的`index.js`文件里编写这个demo的配置.

具体方式为`childrenRouteConfig`数组下配置路由名和描述
